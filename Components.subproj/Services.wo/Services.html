Les services de l'application Demande de Travaux
(version )

 Direct Actions

Le URL pour appeler une "direct action" d'une application est le suivant :
  http://mon.serveur.com/cgi-bin/WebObjects/UneApplication.woa/wa/nomAction?parametre1=valeur1&param2=val2&...

 Accès à la Validation des demandes

Description :

Cette action affiche la page avec la liste des demandes à valider. L'utilisateur doit s'identifier avant d'accéder à cette page. La liste est proposée en fonction de l'identité et donc des droits d'utilisateur dans l'application Demande de Travaux.

URL :

http://.../DT3.woa/wa/DAValidation

Paramètres :

(Cette action ne prend aucun paramètre.)

 Accès à la Création des Demandes

Description :

Cette action permet d'accéder au formulaire de création des demandes restreint uniquement à un service destinataire concret.

URL :

http://.../DT3.woa/wa/DACreation?<parametres>

Paramètres :

Paramètre Présence Description
cs Obligatoire Le code de service. Ceci est un code interne d'un groupe (structure) associée au service.
Dans la base de données, ce code correspond au champ C_STRUCTURE de la table STRUCTURE_ULR.

Exemple :
cs=25

actOrdre Optionnel Le code de l'activité. Ceci est un code interne d'une activite associée au service.
Dans la base de données, ce code correspond au champ ACT_ORDRE de la table ACTIVITES.

Exemple :
actOrdre=1

 Accès à la Création de demandes par email

Description :

Cette action permet d'accéder à la page contenant les adresses email pour la création des demandes. Un message email envoyé à l'adresse donnée déclenche la création d'une demande de travaux dans l'activité correspondante. La liste des activités est donnée avec les adresses email.
Le lien vers cette action est également donné dans la page d'accueil de l'application Demande de Travaux.

URL :

http://.../DT3.woa/wa/DACreationMail

Paramètres :

(Cette action ne prend aucun paramètre.)
